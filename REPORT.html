GEWALT — Storefront

Next.js 14 + Tailwind + Saleor GraphQL

Versión: 1.0.0 Fecha: July 2026 Repositorio: github.com/Jhony33663/Gewalt Branch: storefront-nextjs

1. Resumen Ejecutivo

Storefront profesional para la marca GEWALT streetwear desarrollado con Next.js 14, Tailwind CSS y Saleor como backend de e-commerce. Incluye homepage con video, catálogo con productos reales, selector de talla + color, integración directa a WhatsApp con mensaje pre-llenado, y diseño editorial basado en la dirección visual de la marca.

Características entregadas

  • Hero con video autoplay — Video de fondo con degradado editorial y CTA
  • Catálogo real — Datos desde Saleor GraphQL con precios, imágenes y stock
  • Selector de talla + color — Circulitos de color + tallas con mensaje WhatsApp completo
  • Imágenes oficiales GEWALT — Fotos reales de gewaltoficial.shop
  • WhatsApp checkout — Mensaje pre-llenado con producto, talla, color y precio
  • Diseño editorial — Instrument Serif, Space Grotesk, DM Sans + paleta verde/negro
  • Drops / Colecciones — Poblado desde Saleor collections API
  • Footer profesional — Newsletter, señales de confianza, links organizados

7. Arquitectura Técnica

Stack

  • Framework: Next.js 14 (App Router, Server Components)
  • Styling: Tailwind CSS 3 + CSS custom properties
  • Backend: Saleor (GraphQL API — localhost:8001)
  • Animaciones: Framer Motion
  • WhatsApp: wa.me deep links con mensaje pre-llenado

Estructura de Carpetas

gewalt-storefront/
├── public/                 # assets estáticos (imágenes, video)
├── src/app/               # Next.js App Router pages
│   ├── page.tsx            # Homepage
│   ├── shop/               # Catálogo
│   ├── product/[slug]/     # Detalle + size/color selector
│   ├── about/              # Manifiesto
│   ├── drops/              # Colecciones
│   ├── contact/            # Contacto
│   ├── help/               # FAQ, Shipping, Size Guide
│   └── privacy/, terms/    # Legales
├── src/components/         # UI reutilizable
└── src/lib/                # saleor.ts, whatsapp.ts, product-images.ts
